## Prerequisites
- Xcode (for iOS development and simulator)

## Environment Setup

1. Create a `.env` file in the root of the `mobile` directory.
2. Add your Supabase URL to the `.env` file:
   ```
   EXPO_PUBLIC_SUPABASE_URL=...
   EXPO_PUBLIC_SUPABASE_ANON_KEY=...
   ```
